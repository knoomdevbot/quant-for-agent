## Alpha Hypothesis
- Claim: On public CPI/PPI release dates, after the 8:30 ET release is observable and the regular session opens, unusually large cross-asset opening-gap dispersion among inflation/rates/USD/commodity/equity ETFs may overreact or underreact; a predeclared release-morning gap-dispersion signal can be tested from release-day close to the next 1-3 closes.
- Economic / behavioral intuition: CPI/PPI releases are scheduled macro information shocks. The signal is not a date-only calendar effect; it uses only the market's release-morning opening reaction, which is observable after the market opens and before the close, to test whether cross-asset repricing is excessive or persistent.
- Expected failure mode: Prior local evidence is hostile for CPI/FOMC/PPI event reversal, intraday announcement reversion, overnight/open-gap reversal, real-rate shock fades, inflation rotations, and public-calendar effects. The effect may be generic open-gap reversal/continuation, ETF TSMOM, rates/USD/commodity beta, one-regime dominance, or sparse-event overfit.

## Implementation Plan
- Candidate product pool: Broad liquid U.S.-listed ETFs with inflation/rates/USD/commodity/real-estate/equity exposure: TIP, SCHP, TLT, IEF, SHY, UUP, GLD, DBC, USO, XLE, XLRE, SPY, QQQ plus close substitutes only if coverage requires replacement.
- Product-selection filters: qfa/Alpaca real daily OHLCV coverage, regular open/high/low/close availability, sufficient history for release-event windows, liquidity/dollar-volume suitability, and economic exposure match. No performance-based symbol selection.
- Selected universe: To be fixed by evaluator before performance review from the candidate pool by coverage/liquidity and documented in artifacts.
- Universe-selection rationale / limitations: Cross-asset macro-release reaction requires rates, inflation, USD, commodities/energy, real estate, and broad equity sleeves. Using ETFs is a fast-falsification proxy and cannot observe futures-implied CPI surprise or intraday reaction path.
- Data source: qfa/Alpaca real daily OHLCV only, plus deterministic public CPI/PPI release calendars encoded by rule or compact static calendar notes. No CSV; no `--data-csv`; no daemon; no orders.
- Signal features: CPI/PPI release-day flag; release-day opening gap measured as open vs prior close for each selected ETF; prior-history volatility/z-score normalization; cross-asset gap dispersion; predeclared primary leg must use release-day open and prior history only, not release-day close/high/low/volume.
- Model form: Rule-based event study. Primary leg should be fixed before testing: either fade top-vs-bottom release-morning gap dispersion or continue cross-asset gap direction. Execution/evaluation is release-day close to next 1-3 closes, with clear timing documentation to avoid same-day close feature leakage.
- Parameters / search space: event_type=[CPI,PPI,combined diagnostics]; gap_z_lookback=[60,120,252]; primary_direction=[predeclared fade or continuation]; holding_periods=[1,2,3]; cost_bps_one_way=[5,10]. Do not select the primary direction after performance review.
- Position sizing: Equal-weight sleeves, gross normalized, sleeve caps, cash when too few symbols pass coverage or signal thresholds.
- Risk constraints: Reject if shifted-label/non-release placebos match or beat the primary, if generic gap reversal/continuation explains results, if one year/regime/event type dominates, if p25 Sharpe is materially negative, or if redundancy to macro/TSMOM/gap baselines is high.
