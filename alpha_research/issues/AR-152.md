## Alpha Hypothesis
- Claim: Public EDGAR N-PORT/NPORT-P fund holdings disclosures, traded only after EDGAR acceptance/public timestamp and conservative publication lag, may identify crowded liquid common-stock or ETF holdings increases/decreases that create delayed price pressure, reversal, or drift in tradable U.S. equities/ETFs.
- Economic / behavioral intuition: Delayed public disclosure of fund portfolio changes can reveal crowded institutional demand/supply, changing ownership breadth, or fund-family pressure that may not be fully reflected immediately at disclosure time.
- Expected failure mode: Holdings are stale by publication; mapping and fund-series continuity fail; signal reduces to mega-cap momentum/reversal, sector momentum, 13F filing-calendar attention, fund-family concentration, or generic price/volume pressure.
- Falsifier: Reject before performance if a reproducible timestamp-safe EDGAR event table with conservative mapping cannot be built. Reject after performance if shifted/shuffled holdings changes, sector/momentum/reversal, AR-130 filing-calendar, or concentration controls match or dominate the primary signal.

## Implementation Plan
- Candidate product pool: All public EDGAR NPORT-P/N-PORT filings over the qfa/Alpaca overlap; U.S.-listed common stocks and ETFs with filing-provided ticker or conservative CUSIP/name mapping and liquid Alpaca daily bars.
- Product-selection filters: EDGAR acceptance/public timestamp present; report period present; fund/series identifiers stable enough for prior-holding comparison; holding has common-stock or ETF classification; reliable ticker/CUSIP mapping; Alpaca daily-bar coverage; liquidity and event-dispersion gates.
- Selected universe: To be selected only after source gate. Primary sleeve should use mapped liquid U.S. common stocks; ETF sleeve is diagnostic. Require at least 150 mapped liquid symbols and preferably thousands of fund-security change observations, with no single issuer, fund family, sector, or year dominating.
- Universe-selection rationale / limitations: Fund holdings disclosures are broad ex ante, but stale reporting/public lag and mapping/survivorship issues are severe. Ambiguous mappings and non-equity/fixed-income/derivative holdings should be dropped rather than guessed.
- Data source: SEC EDGAR daily indexes, NPORT-P/N-PORT filings and XML holdings, using EDGAR acceptance timestamp/public availability for trading eligibility; qfa/Alpaca real daily OHLCV for returns only if source gate passes. No CSV market data, no daemon, no orders.
- Signal features: Same-fund/series/security value change, balance/share change, pctVal change, fund-family normalized change, cross-fund breadth of increases/decreases, large positive/negative holdings-change shock buckets.
- Model form: Source/event-table feasibility scout first; rule-based event drift/reversal ranker only if source gate passes.
- Parameters / search space: 5 and 20 trading-day post-publication holding periods; positive vs negative holdings-change shock buckets; common-stock primary sleeve; ETF diagnostic sleeve; 10 bps primary one-way costs with 5/20 bps sensitivity.
- Position sizing: Equal-weight or rank-weight event portfolio with gross exposure <=1.0, max issuer/fund-family/sector caps, cash when no valid events.
- Risk constraints: Trade only after EDGAR acceptance/public timestamp mapped to next liquid session; no report-period-date trading; no aggressive ticker/CUSIP inference; exclude ambiguous debt/options/swaps/non-U.S. holdings from primary sleeve.

## Evaluation Plan
- Random-period protocol: Source/event-table gate first. If passed, run random event windows and leave-year diagnostics over the qfa/Alpaca overlap.
- Metrics: Source coverage, parser success rate, mapped-symbol count, event count, event concentration, median/p25/worst random-window Sharpe, positive-window rate, annualized return/volatility, max drawdown, turnover, hit rate, activation rate, cost sensitivity.
- Costs/slippage assumptions: Primary 10 bps one-way; 5/20 bps sensitivity.
- Orthogonality checks vs alpha library: AR-130 SEC filing-calendar attention, generic momentum/reversal, sector momentum/reversal, raw price/volume pressure, shifted publication dates, shuffled holdings changes, fund-family concentration, 13F filing-calendar proxy.
- Acceptance threshold: Source gate passes; post-cost median random-window Sharpe >0; p25 not materially negative; positive-window rate >=55%; 20 bps stress does not collapse; primary beats shifted/shuffled, momentum/reversal, sector, raw pressure, and 13F-calendar controls; max relevant correlation <=0.60 hard / <=0.50 preferred; no single issuer/fund family/sector/year dominates.
