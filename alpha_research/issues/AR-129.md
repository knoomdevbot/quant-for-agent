## Alpha Hypothesis
- Claim: Liquid U.S. large-cap single names may exhibit stock-specific month-of-year return persistence: names with historically strong returns in the upcoming calendar month, measured only from prior years/months available before the rebalance date, outperform names with historically weak same-month returns over the next month.
- Economic / behavioral intuition: Recurrent firm-specific seasonal flows, investor attention cycles, fiscal/product/calendar seasonality, tax/rebalancing patterns, and delayed behavioral anchoring can create repeatable month-of-year cross-sectional return tendencies. This differs from broad turn-of-month ETF timing because the signal is stock-specific and cross-sectional, not a market-wide date window.
- Expected failure mode: Effect is weak in the short Alpaca/qfa sample, dominated by generic momentum/low-vol/sector tilts, one calendar month or mega-cap tech cluster, or erased by monthly turnover costs. Static current large-cap universes introduce survivorship bias.

## Implementation Plan
- Candidate product pool: Fixed ex-ante liquid U.S. large-cap/common-stock pool of roughly 80-150 names if Alpaca/qfa coverage allows, excluding ETFs/funds/REITs where feasible. Sector ETF proxies SPY, QQQ, XLB, XLC, XLE, XLF, XLI, XLK, XLP, XLRE, XLU, XLV, XLY are diagnostics/residualization controls only.
- Product-selection filters: Alpaca/qfa real daily OHLCV availability; sufficient history for lagged same-month observations; high median dollar volume and minimum price; static sector mapping if used; no return-based symbol selection.
- Selected universe: To be fixed by evaluator before performance review using only coverage, liquidity, common-stock classification, and optional sector mapping. Record selected symbols, final universe size, and limitations.
- Universe-selection rationale / limitations: The mechanism requires a broad single-name pool to distinguish firm-specific seasonal patterns from market calendar seasonality. Static current membership is a survivorship-biased fast-falsification approximation.
- Data source: qfa/Alpaca real daily OHLCV only; no CSV; no `--data-csv`; no daemon; no orders.
- Signal features: Monthly close-to-close returns derived from daily bars; for each rebalance date compute each stock's lagged average/median return for the upcoming calendar month using only prior years/months; require minimum historical observations; optional sector-neutral/residual version; controls for 12-month momentum and realized volatility only as baselines/diagnostics.
- Model form: Rule-based cross-sectional monthly allocator.
- Parameters / search space: same_month_history_years=[3,5,all_available_lagged]; statistic=[median,trimmed_mean]; rebalance=[last_trading_day_prior_month,first_trading_day_month]; holding_period=one_calendar_month; portfolio=[long_short_top_bottom_quintile,long_only_top_quintile_underweight_bottom]; risk_controls=[sector_neutral,sector_capped,vol_scaled]; cost_bps_one_way=[5,10,20].
- Position sizing: Equal-weight or equal-risk buckets, gross exposure <= 1.0, per-name cap, sector cap or sector-neutral diagnostic, no leverage beyond normalized research weights.
- Risk constraints: Reject if performance depends only on January/December, one year, one month, or one sector; reject if generic 12-month momentum/low-vol/sector baselines explain results.

## Evaluation Plan
- Random-period protocol: Use Alpaca/qfa real daily OHLCV, compact evaluator, monthly replay over available 2020-present history or maximum coverage. Because same-month history needs warmup, start performance only after minimum lagged history is present. Run at least 20 random contiguous windows if enough months are available; otherwise explicitly mark data-limited and use rolling leave-year/month-out diagnostics.
- Metrics: After-cost Sharpe at 10 bps one-way, median/p25/worst random-window Sharpe, positive-window rate, annualized return and volatility, max drawdown, monthly turnover, active name count, sector concentration, month/year contribution attribution.
- Costs/slippage assumptions: Primary 10 bps one-way cost, plus 5/20 bps sensitivity.
- Required ablations: Generic 12-month momentum cross-sectional baseline; low-vol/defensive factor baseline; sector-neutral vs raw; randomized month labels within each year; prior-month-only return rank placebo; equal-weight large-cap universe baseline; exclude January/December; month-by-month attribution; long-only/underweight variant; 5/10/20 bps cost sensitivity.
- Orthogonality checks vs alpha library: Compare/proxy against AR-010 low-vol quality, AR-001 cross-sectional momentum lineage, AR-006/021/041 turn-of-month ETF seasonality, AR-038 liquidity-seasonality balance-window, AR-107 residual momentum, AR-113 idiosyncratic vol normalization, and broad SPY/QQQ/sector ETF factors where retained streams or proxy correlations are available.
- Acceptance threshold: Positive 10 bps after-cost median random-window Sharpe, p25 Sharpe not materially negative, positive-window rate >= 55%, no single month/year/sector dominance, randomized-month placebo percentile >= 0.85, and max relevant proxy/library correlation preferably <= 0.60.
