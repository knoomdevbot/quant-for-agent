## Alpha Hypothesis
- **Claim:** Equity momentum performs differently under bond/gold/USD stress signals; a simple regime switch may improve drawdown-adjusted returns.
- **Economic / behavioral intuition:** Basic, interpretable signal intended as a seed branch for evolutionary alpha research.
- **Expected failure mode:** Costs, crowding, regime shift, parameter fragility, or redundancy with existing price-based alphas.
- **Falsifier:** Median random-period Sharpe <= 0 after costs, strongly negative lower-tail windows, or high redundancy with accepted alpha library without improvement.

## Evaluation Plan
- **Random-period protocol:** Smoke test first, then 30 random windows if enough data exists; otherwise mark data-limited and run available-window smoke.
- **Metrics:** Sharpe, annualized return, annualized volatility, max drawdown, win rate, period count, turnover if available.

- **Orthogonality checks vs alpha library:** Compare return stream to accepted/watchlist models when available; otherwise mark library empty.
- **Acceptance threshold:** Positive median random-window Sharpe after costs, no obvious leakage, interpretable parameters, and not redundant with stronger accepted alpha.
